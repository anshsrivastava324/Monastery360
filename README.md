🏔️ Monastery360 – Virtual Monastery Experience

🌐 Live Demo: https://monastery360vr.netlify.app/

Monastery360 is an immersive digital platform designed to digitize and showcase monasteries of Sikkim through interactive maps and 360° virtual experiences. This project was developed as part of the Smart India Hackathon (SIH) – Problem Statement ID: 25061.

🚀 Features
🌍 Interactive Map using Leaflet.js
🏯 360° Virtual Tours of Monasteries
🧭 Tourism-focused exploration experience
🎨 Clean, modern, and responsive UI
🛠️ Tech Stack

Frontend:
HTML5, CSS3, JavaScript

Libraries & Tools:
Leaflet.js (for maps)
360° Panorama Viewer (pannellumJS)

Deployment:
Netlify
